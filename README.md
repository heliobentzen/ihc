## 🎯 Como usar este material

Cada módulo contém:
- 📖 **Conceitos** explicados de forma clara e objetiva
- 🛠️ **Práticas** com exercícios hands-on
- 💡 **Exemplos reais** de produtos digitais conhecidos
- ✅ **Checklists** prontos para usar nos seus projetos
- 🔗 **Referências** curadas para aprofundamento

---

## 🚀 Pré-requisitos

Nenhum conhecimento prévio de design é necessário. O material é adequado para:
- Estudantes de Sistemas de Informação, Computação e áreas afins
- Desenvolvedores que querem entender melhor o lado do usuário
- Qualquer pessoa curiosa sobre como criar interfaces melhores
